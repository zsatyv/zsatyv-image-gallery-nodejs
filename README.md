## Installation

1. **Install dependencies:**
```bash
npm install
```

2. **Initialize database:**
```bash
npm run init-db
```

3. **Start the server:**
```bash
# Development mode (with auto-reload)
npm run dev

# Production mode
npm start
```

4. **Visit:** http://localhost:3000

That's it! No database server installation needed - SQLite just works! ✨
